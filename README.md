## Opciones
### Primera opción:
Ingresa un número de dígitos y se calcula según la secuencia Fibonacci.

### Segunda opción:
Se pregunta por un número y se verifica que sea capicúa.

### Tercera opción: 
Se pregunta por un número y se suma por todos sus divisores. Dependiendo de el resultado se muestra si es o no un número perfecto.

### Cuarta opción:
Se ingresa un valor inicial y final, se filtra los números que no sean primos y solo se imprimen los números primos.

### Quinta opción:
ingresa un número aleatorio y el programa dira si es o no un número primo.

### Sexta opción:
Solicita la cantidad de factorización y luego devuelve el resultado final.

### Septima opción:
Pide 2 números, reconoce sus divisores y luego compara el de ambos números y devuelve divisor en común que tengan ambos números.

### Octava opción:
Termina con el proceso de todo el bucle y cierra el menu.
